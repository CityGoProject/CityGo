📖 Proje Nedir?

Farklı ulaşım türlerini (✈️ uçak, 🚆 tren, 🚌 otobüs) tek bir platformda birleştiren Java tabanlı full-stack web uygulaması.

👤 Yolcu Özellikleri

  • Kayıt & giriş (e-posta/şifre)
  • Sefer arama ve filtreleme
  • Görsel koltuk haritasından seçim
  • Bilet alma, görüntüleme, iptal

🔧 Admin Özellikleri

  • Sefer ekleme, güncelleme, silme
  • Kullanıcı ve bilet listeleme
  • Dashboard istatistikleri
  • Veriyi JSON / CSV dışa aktarma
